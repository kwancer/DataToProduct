# DataToProduct

This repository contains the code for the Data to Product coursework by Krzysztof and Erietta.

## Required R Packages

Install all dependencies with:

```r
install.packages(c(
  "shiny",
  "leaflet",
  "jsonlite",
  "later",
  "dplyr",
  "lubridate",
  "slider",
  "tibble",
  "mgcv"
))
```

## Running the App

1. Clone the repository.
2. Install all required R libraries.
3. Open `app/app.R` in RStudio.
4. Click **Run App**.

The application should run without further setup.

## Model Information

The `model` folder contains the scripts used for data cleaning, merging, model training, and testing.  
The original raw datasets are not included due to their size, except for the final dataset used during model training.  
Links to the data sources are available in the accompanying report.
